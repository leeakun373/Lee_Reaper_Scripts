## 🔧 自定义调整

### 调整透明度

在 `styles.lua` 中修改：

```lua
-- 更透明
sector_default = {150, 150, 150, 80}  -- alpha 从 120 改为 80

-- 更不透明
sector_default = {150, 150, 150, 160}  -- alpha 从 120 改为 160
```

### 调整子菜单背景

在 `list_view.lua` 中修改：

```lua
-- 更透明
reaper.ImGui_SetNextWindowBgAlpha(ctx, 0.7)  -- 从 0.85 改为 0.7

-- 更不透明
reaper.ImGui_SetNextWindowBgAlpha(ctx, 0.95)  -- 从 0.85 改为 0.95
```

### 调整边框粗细

在 `wheel.lua` 中修改：

```lua
local border_width = 2.0  -- 从 1.5 改为 2.0（更粗）
```
